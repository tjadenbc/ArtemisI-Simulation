What this simulation does. Each trial samples ascent engine-out / ignition events, engine + insertion dispersions, orbit-determination nav-knowledge and burn-execution errors, and the sourced failure model, then runs the full uncrewed Artemis I mission with real-physics ODE integration — pad to splashdown as ONE continuous trajectory (SLS ascent → ICPS TLI → outbound OTC correction chain → powered lunar flyby → DRO insertion / 6-day stay / departure → return powered flyby → skip entry → splashdown). Uncrewed → success = SLS delivers Orion AND Orion returns & splashes down intact (there is no crew-survival model). When a mission fails, the cause is recorded as a mission_failure value; the Failure Analysis below explains each in plain language. Failure-mode probabilities are estimates (heritage reliabilities + NASA PRA + foreseeable-problem analysis) — sourced from NASA / AAS references.

Mission Outcome

Cumulative Phase Survival

One bar per mission timeline phase (the same phases as the Phase Timing table below). Each bar is the fraction of trials still on-track at the END of that phase — i.e. whose mission was not lost in it or any earlier phase — so the bars step down monotonically and the final bar equals the mission-success rate. Phase-specific losses land in their own phase; the cross-cutting spacecraft-systems failures are placed in the phase they actually occurred in (by their recorded ground-elapsed time).

Known Limitations

1. Failure-mode probabilities are ESTIMATES. Artemis has flown twice (I 2022, II 2026), both successful — no statistical failure base exists. Inputs are heritage component reliabilities + NASA PRA targets + foreseeable-problem analysis (sourced from NASA / AAS references). The success number is only as good as these. The dominant failure here is `esm_systems_failure` at 1.6%.
2. Navigation is a geometry-derived OD covariance, not a live sequential filter. The nav covariance is built EMERGENTLY from DSN station geometry + tracking-arc physics via an STM-LinCov filter (`ENABLE_OD_FILTER`, default-ON) — it replaced the hand-set calibrated-covariance stand-in and delivers the as-flown OD-knowledge envelope, so DELIVERY PRECISION is now faithful: recovery-zone displacement ~1.15 km median, splash-miss ~4.9 km median. The remaining approximation is that it is a LINEARIZED-COVARIANCE (LinCov) model, not a live sequential EKF re-solving state from synthetic raw range/Doppler/ΔDOR observations. Guidance corrections are applied on the estimated state; each is flown to convergence, not re-tracked continuously.
3. Entry is a PredGuid drag-tracker; the shallow-corridor deep-dip velocity is unmatched. A PredGuid-class drag tracker — bank modulation + closed-loop crossrange bank reversals + a lob-lift downrange trim + an extended-skip long-range branch. Nominal peak ~4.4 g ≈ real Orion ~4 g, and it reproduces the real TWO-PULSE skip; splash accuracy is MATCHED (~4.9 km median ≈ real ~4.7). Residual (the HUNTEST wall): at Orion's shallow −5.9° corridor the first dip bottoms ~10 km/s vs the real deep ~8–9, and matching that depth needs a steeper EI → ~10 g overstress — so peak-g OR deep-dip velocity, not both. Atmosphere is static USSA-76 with per-trial ±6% density dispersion + chute-phase wind drift (the ~3.5 km splash floor); no time-varying weather.
4. The RPF return premium is structural; the reference DRO is now ephemeris-native. The reference DRO is an EPHEMERIS-NATIVE ballistic arc of the sim's own full force model (fit to the as-flown OEM DRO segment, node RMS ~79 km, the real elliptical 66,000–96,000 km geometry — the CR3BP map is retired). Insertion is now essentially exact: OPF+DRI ~289.5 vs the real 289.2 m/s (DRI ~110.2 ≈ real 110.6), arrival residual ~5 km, max-Earth 432,197 ≈ real 432,194 km, and DDP ~144.6 ≈ the DESIGN departure 145.2. The one honest residual: RPF ~371 m/s vs the as-flown 292.9 — a multi-seed basin search proved the ~293-class solution family UNREACHABLE from the sim's state at the real epochs, so the premium is structural to the two-burn/fixed-corridor return formulation, not a reference artifact. The return is the epoch-committed PLANNED trans-earth branch (100% of successes), not a fallback. The real orbit's small OM-1/OM-3 maintenance burns remain unmodeled (the arc's DDP-end offset vs the OEM ≈ OM-3's 13.2 m/s).
5. Nominal targets the PLANNED zone; the as-flown Guadalupe splash is a dispersion scenario. Per the as-planned doctrine the nominal aims the PLANNED recovery zone (off San Diego, ~32°N, 118°W), matching the flight plan (timeline 25.46 d (≈ Artemis I's 25.45 d)). Artemis I actually splashed ~350 km south near Guadalupe Island (~29°N) after an in-mission WEATHER RETARGET — a discrete operational move beyond the fleet's natural dispersion, so it is reproduced as a named re-aim scenario, not the nominal. The old Indian-Ocean fallback + region offset are CLOSED. Separately, mid-mission burns (OTC/DRI/DDP/RPF/RTC) are impulsive Δv with execution-error draws (launch/PRM/TLI/OPF are finite). The outbound path now sits ~960 km from the as-flown OEM (the Phase-C target re-derive on the current force model closed the prior ~2,091 km staleness residual).
